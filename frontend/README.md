# FinSight Frontend

React + Vite + TypeScript client for FinSight.

## Develop

```bash
npm install
npm run dev      # http://localhost:5173 (proxies /api → http://localhost:8000)
```

## Build

```bash
npm run build
```

## Roadmap (M5)
- Chat UI with token streaming over WebSocket
- Document upload + ingestion progress
- Task panel showing concurrent background tasks
- Citation viewer (deep-link to source page on Cloudinary)
